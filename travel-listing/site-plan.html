Overview

Purpose

Users can create and manage travel-related lists. They can customize the list types by interest, region, or theme and check mark places visited. Some examples would be “Dream Destinations”, “Coastal towns”, “European cities”, “National Parks”, and “Historic Sites.”

Audience

This page is suitable for all ages but targeted to travel enthusiasts. Some examples of users would be:

  • College students planning budget or backpacking trips
  • Young professionals planning future travel ideas
  • Families tracking places visited together
  • Retirees/older couples planning larger trips and checking off bucket-list destinations

Dynamic elements

People can make lists and save them. With JavaScript we can add items to the list to display them and save them on local storage. The lists can be sorted and filtered for searching. There can be some pictures that are displayed randomly on the home page each time it refreshes.
